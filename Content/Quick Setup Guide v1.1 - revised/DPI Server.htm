DPI Server

Deploy the server using the vSphere client

  1. In the vSphere client, navigate to File > Deploy OVF Template…
  1. Enter the empow-dpi-universal.ova file location.

  1. Choose a storage provisioning method, as for the Cassandra and SST servers.
  1. Configure the VMware vSwitch to allow all VLAN IDs, and to enable promiscuous mode, in order to deliver all the mirrored traffic it receives to the empow DPI server.
